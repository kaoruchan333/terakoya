# 中庸思考 コア知識
出典：メンターのための中庸思考（人生デザイン構築学校）

## 中庸とは
- 人間の欲求ではなく、神性で生きること
- 中庸のゴール：**全てのネガティブに心からの感謝ができる**

## 世界の真実（法則）

**真実4：自分編**
- 全ての人は、人間が持ち得る全ての特性を持っている

**真実5：**
- 自分の特性は、良くも悪くなく、一つの特性にすぎない
- 短所という特性はない。長所という特性もない

**真実6：**
- 誰もが同じように、何をしてもしなくても価値ある存在
- 存在として、優劣はない

**真実7：**
- 理想は、現実的でない幻想
- 理想は、物事のプラス面だけを見て「起こったらいいな」を掲げている状態

**真実8：**
- 恐れと不安は起こり得ない幻想・自分が作り出した妄想

**真実11：**
- 批判と賞賛は、常に半分ずつ
- 理解してくれる人とくれない人は半分ずつ
- 応援してくれる人とくれない人は半分ずつ

**真実12：**
- この世界は、どこで切っても常に、正負のバランスが取れている
- 自分、1対1、家族、コミュニティ、国、世界

**真実13：**
- 成長は、試練と支援の狭間で起こる
- ネガティブが存在しなければ、ポジティブも存在しない

**真実14：**
- 認知が変わらない限り、同じ課題に直面し続ける
- 対消滅していない感情は「フレームワーク（信念体系）」として潜在意識に蓄積される

**真実15：宇宙編**
- エネルギーは常に流れていて、終わりも始まりもない（万物流転の法則）
- 「変化しないことが安全」は、幻想

**真実17：**
- 足りないものは、何もない
- ありのままで完璧
- 変わる必要はない。成長あるのみ

## 認知→思考→感情の流れ
出来事 → 認知（ポジ/ネガ） → 思考（ポジ/ネガ） → 感情（ポジ/ネガ）

感情がポジティブに傾いている時 = マイナス側が見えていない状態
→ 反対側（マイナス）を見に行く → 中庸に戻る

## 中庸思考を極める4段階
01. 自己変容
02. 全体性の発揮
03. 自己超越
04. 悟り

## 自己変容とは
- 真実を認知できることが増えていくこと（視点が増える・視座が上がる）
- 受容できることが増えていくこと
- 時間軸が長くなること
- 精神性が向上すること＝自我が無くなっていくプロセス

## 全体性とは
「他人にどう見られているか」を気にして身につけている「仮面」を脱ぎ捨て、
ありのままの自分（全体 as a whole）として存在すること。
→ 活力・創造性・熱意が発揮される

## 自己超越とは
- 自我が無くなり、悟りに向かう
- 他人の喜び＝自分の喜び
- 搾取から循環へ・競争から共創へ・所有よりシェアへ
- 利他＝利己（利他≠自己犠牲）

## コーチングへの応用

### 認知の偏りゲーム
私たちは事実と異なる認知をし、それに基づいて感情的になり反応する。
相手もまた相手の認知で反応する → 認知の偏りゲームが起きる

### コーチができること
1. **自分の在り方で全体性を示す** - ネガティブな特性も曝け出す・混沌をダメとしない
2. **相手を受け入れる** - 傾聴・共感・ジャッジなし。認知と感情を真実と思い込まない
3. **相手に働きかける** - 真実に気づくための質問。一歩ずつ、階段の高さを合わせる

### フィードフォワード（フィードバックではなく）
- ジャッジなし・アドバイスなし・引き上げなし
- 支援と励まし
- 真実に自分で気づく質問（小さい階段）
- **上からの引き上げではなく、隣での伴走**

### コーチが覚えておきたい真実
「質問の質が人生の質を決める」
どんなに複雑な課題でも、やることは同じ。
真実に気づくための質問を忍耐強く、一貫してし続ける。

## 愛と感謝・中庸の状態の本質

### 2つの世界を分けて理解する

**目に見える世界（物理・構造）：**
- 矢印（→）の方向がある。順番・時間・方向がある
- コップに水を注いで飲む、感謝する→届く→返ってくる
- 映画のように始まりと終わりがある構造の世界

**目に見えない世界（精神性）：**
- 全てが同時にある。矢印がない
- ありのままが見えた瞬間、時間も空間も消える
- 夕日の前で涙した瞬間、赤ちゃんの顔を見た瞬間がそれ

→ この2つを混ぜずに分けて語ることが重要。どちらの視点から話しているかを明示する。

### 感動とは何か
感動 = 矢印のある世界に生きながら、矢印のない世界に触れた瞬間。
これが愛と感謝・中庸の状態。「良い」「悪い」という二元論の言葉ではなく「感動」「インスピレーション」が正確な表現。

### 認知がバランスされた瞬間（対消滅）
- 感情がプラスに傾く → 反対側（マイナス）を見に行く
- 感情がマイナスに傾く → 反対側（プラス）を見に行く
- 両方見えた瞬間 → 認知の偏りが消える（対消滅）
- ありのままの事実・真実が見える → 全てが愛だったと分かる

### 愛と感謝はたどり着くものではない
「たどり着く」は矢印の世界の言葉。中庸の状態は目的地ではない。

私たちはこの体を持ち、時間と空間の中で生きているから感情は揺れる。
プラスにもマイナスにも揺れ、反対側を見る繰り返しの中で、
プラスもマイナスも判断も評価も**全て卒業されていく**。
卒業されたその先に残るのが愛と感謝。

**たどり着くのではない。本来の状態が、立ち上がる。**
