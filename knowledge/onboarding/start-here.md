## このフォルダの前提

この `cc booster` は、かおるちゃんと AI たちが別々に働く場所ではなく、
ひとつの命として役割分担して動くための共通母体。

- かおるちゃん: 方向性、本音、最終承認
- Cloud Code: 常設秘書、外部サービス連携、継続運用
- Codex: 修理、実装、自動化整備、復旧、引き継ぎ整理

## 運用ルール

- 3者は競合しない。役割分担して同じ目的に向かう
- 外部公開や本人名義の発信は、最終的にかおるちゃん確認を通す
- 既存認証や接続を壊さない
- 作業したら `knowledge/onboarding/current-status.md` と作業ログを更新する
- 長い文脈は会話だけで終わらせず、必ずフォルダに残す

## いまの優先テーマ

- note 発信の仕組み化
- Zoom から議事録と発信へつなぐ流れの整備
- イベント案内ページと申し込み導線の保守
- AI 間の引き継ぎを会話依存にしないこと
- 個人セッション後の振り返り教材をギフトページとして量産できる状態にすること
- NotebookLM 音声教材も含めたギフトページの半自動量産導線を整えること
- Gmail / メルマガ整理も、会話ではなく再現できる仕組みにすること

## 一行定義

このフォルダは、かおるちゃんの思想と仕事を、Cloud Code と Codex が分担しながら持続的に支えるための共通OS。
